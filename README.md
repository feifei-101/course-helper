#  狒狒助手

Edge 浏览器扩展

---

## 📥 下载安装

### 1. 下载项目代码

方式一：下载 ZIP（推荐）
1. 打开 https://github.com/feifei-101/course-helper
2. 点击绿色 Code按钮 → Download ZIP
3. 解压到任意文件夹

方式二：Git 克隆
`ash
git clone https://github.com/feifei-101/course-helper.git
`

### 2. 安装到 Edge

1. 打开 Edge，地址栏输入 edge://extensions/ 回车
2. 打开左下角 "开发者模式"
3. 点击 "加载解压缩的扩展"
4. 选择解压后的 course-helper/ 子文件夹（里面直接包含 manifest.json）
5. 工具栏出现 🦍 图标即安装成功

## 🔑 配置 API Key

AI 功能需要 API Key。

### 获取 Key

1. 打开API网站
2. 菜单 API Keys → 创建 API Key
3. 复制生成的 sk- 开头的密钥

### 填入扩展

1. 点击工具栏 🦍 图标
2. 点击 行展开输入框
3. 粘贴 Key →  保存→  测试
4.  连接成功即可使用

## 功能使用

### ▶️ 自动刷课

进入页面，扩展自动检测视频并播放，完成后自动跳转下一节。

- Alt + Shift + A — 快捷键开关自动连播
- Alt + Shift + P — 显示/隐藏浮动面板

### ⏩ 倍速控制

弹出菜单或浮动面板中切换 1x / 1.5x / 2x，设置后全局生效。

### AI 搜题

选中文字 → 右键 → "AI搜题"，打开新标签页查看 AI 回答。

### 答题
进入答题页面自动识别题目并调用 AI 作答：
- 支持单选、多选、判断、填空、简答
- 超过 5 道题分批完成，批间休息 2~5 分钟
- 自动检测提交状态，避免重复提交

状态颜色：
| 颜色 | 含义 |
|------|------|
| ⚫ 灰色 | 就绪 |
| 🟢 绿色 | 播放视频中 |
| 🟡 黄色 | 等待任务完成 |
| 🟣 紫色 | 自动答题中 |
| 🔵 蓝色 | 跳转下一节 |
| 🔴 红色 | 出错 |

弹出菜单和浮动面板显示当天：视频 N · 答题 N · 完成 N，数据存在本地，每日自动归零。

## ❓ 常见问题

加载扩展提示"清单文件缺失"？
选错文件夹了，确保选的是内含 manifest.json 的 course-helper/ 子文件夹。

进入课程没反应？
确认网址  刷新页面 → 检查开关是否开启 → 检查 API Key 是否配置。

视频一直"就绪"？
部分 iframe 视频需手动点一次播放按钮，之后扩展会接管。

API Key 测试失败？
检查是否 sk- 开头、有无多余空格、网络能否访问 、账号余额是否充足。
